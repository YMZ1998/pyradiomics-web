Overview

Step-Based Workspace

Each workflow stage has its own page with a focused form, a result area, and the related preview panel. The layout is optimized for desktop screens and avoids mixing unrelated controls on one page.

Validation Review manifest structure, image geometry, and mask consistency.
Feature Extraction Generate radiomics features and inspect exported outputs.
Selection And Training Reduce the feature set and compare model performance in separate workspaces.

Workflow

Select A Step

01 Validate Input Data

Check manifest completeness, geometry alignment, and mask validity.

 02 Extract Features

Run radiomics extraction using the selected parameter file.

 03 Select Features

Filter low-value and redundant features before model training.

 04 Train Models

Review cross-validation metrics and compare candidate models.

Load a trained model, generate predictions, and compute evaluation metrics.

 All Run Full Pipeline

Execute the full workflow from data validation through training.

Guidance

Recommended Sequence

1. Validate

Confirm the dataset is structurally ready before running downstream analysis.

2. Extract

Generate `features.csv` and use that output in the following steps.

3. Select

Reduce the feature space and export `selected_features.csv`.

4. Train

Evaluate model performance using the selected feature table.

5. Predict

Reuse a saved model for inference and optional evaluation on labeled data.
